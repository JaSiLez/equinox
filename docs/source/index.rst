equinox Documentation
==============================

.. image:: equinox-logo.png
  :width: 100
  :alt: equinox Logo
  :align: left

**Equinox** is an open source platform for collecting and reporting risk and sustainability information about project finance. Equinox implements / integrates the following regulatory / industry standards: BIS/EBA Standardized Credit Risk assessemnt of Project Finance; Equator Principles; PCAF Emissions Accounting and Reporting


* Author: `Open Risk <http://www.openriskmanagement.com>`_
* License: MIT
* Domain Documentation: `Open Risk Manual Project Finance <https://www.openriskmanual.org/wiki/Category:Project_Finance>`_
* Development Website: `Github <https://github.com/open-risk/equinox>`_
